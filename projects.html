Priority Alert Scheduler

Problem: High priority cases are not distrubuted equally to analysts.

Solution: Create an automated system that tracks high priority alerts, adds them to cases, and assigns those cases to analysts based on priority.

I am planning on using the .NET framework for this project. I believe the task of weighting users based on different factors including schedule should be the most interesting part of the program.

Analyst Tools

Problem: The Elastic Discover tool allows a user to search and filter for certain data, but does not provide a convinent way to pivot to searches that utilizes that information.

Solution: Create a Python based, text manipulation tool that provides the results of Discover queries in many useful formats including new queries.

The "sort" program is one of the Analyst Tools designed to take columns copied directly from Elastic Discover and format the values in various ways that are useful to an analyst. Todos include parsing the data from input.txt into a data structure rather than manipulating it directly in the processColumn function. Error handling for incorrect formatting also needs to be implemented.
